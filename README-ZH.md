## 如何使用

1. 克隆此项目至本地
2. 使用 npm / yarn / pnpm 初始化项目
3. 打开 "src/main.tsx" 文件，编写您的脚本
4. 运行 npm 脚本 `rollup -c`，此时会在 `dist` 文件夹中生成一些文件
5. 现在您可以在dist文件夹使用 `ExtendScript Debugger` 运行您的 Photoshop JSX 脚本
